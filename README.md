# Tabletop Encounter Tracker

## Overview

Tabletop Encounter Tracker is a Java-based application designed to assist tabletop RPG players and Dungeon Masters in managing combat encounters. The application provides a simple and efficient way to track initiative order, hit points, and key character statistics during gameplay.

## Features

* Add and manage player characters and enemies
* Track initiative order for turn-based combat
* Update and monitor hit points (HP)
* Display key stats such as Armor Class (AC)
* Real-time encounter updates

## Technologies Used

* Java
* SQLite (if applicable)
* Visual Studio Code

## How to Run

1. Compile the program:
   javac -d bin src/*.java

2. Run the program:
   java -cp bin App
